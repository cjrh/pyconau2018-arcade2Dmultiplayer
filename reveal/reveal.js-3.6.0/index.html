tip #1: use a vector class

Use the one in pymunk


                    >>>from pymunk.vec2d import Vec2d
                    >>> v = Vec2d(1, 2)
                    >>> v.x
                    1
                    >>> v.y
                    2
                    >>> v + 100 * v
                    Vec2d(101, 202)

tip #2: saving and loading game data

Use the one in pyglet!

(pyglet gets installed when you install arcade)


                import os
                import pyglet.resource

                dir = pyglet.resource.get_settings_path('MyGame')

                if not os.path.exists(dir):
                    os.makedirs(dir)

                filename = os.path.join(dir, 'highscores.txt')
                with open(filename, 'wt') as f:
                    f.write(new_scores)
